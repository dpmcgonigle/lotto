## Installation ##
Steps:
- Create and activate virtual environment with python 3.11
```
$ conda create -n lotto python=3.11 -c conda-forge
$ conda activate lotto
```
- Install lotto repository
```
$ git clone 
$ cd lotto
$ pip install .
```
